About This Project

Project 2 — Amazon Rekognition Image Analysis Pipeline

🏗️ Architecture

  • Amazon S3 — Image storage & static hosting
  • AWS Lambda — Serverless compute
  • Amazon Rekognition — AI/ML vision
  • API Gateway — REST API
  • Terraform — Infrastructure as Code
  • CloudWatch — Monitoring & logging

AI Capabilities

  • DetectLabels — Objects & scenes
  • DetectFaces — Facial analysis
  • DetectText — OCR text extraction
  • Confidence scoring on all results
  • Supports JPEG and PNG formats
  • Up to 5MB image size

Tech Stack

  • Python 3.10 — Lambda runtime
  • Boto3 — AWS SDK for Python
  • Terraform 5.0 — IaC provisioning
  • HTML/CSS/JavaScript — Frontend
  • AWS Free Tier — 5000 analyses/month
